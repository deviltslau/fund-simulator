# 基金模拟盘 · 投资复盘工具

可回溯任意历史日期的基金组合模拟与多策略回测工具。真实历史净值 + VIX 恐慌指数择时 + 自动交易生成，**纯静态网页，双击 `index.html` 即用**，无需任何后端或安装。

## ✨ 功能

- **可回溯任意历史日期**：拖动时间滑块回到过去任意交易日，查看当日持仓、市值与净值。
- **回溯交易 / 持仓管理**：在历史某天下达买入/卖出，逐日回测份额、成本价、市值与已实现盈亏。
- **真实历史净值**：内置 13 只基金（沪深300 基准 + A股宽基/行业 + 纳斯达克场内外）真实完整历史单位净值，另支持 CSV 导入与应用内尽力抓取。
- **净值 / 组合走势图**：组合曲线采用**时间加权收益指数(TWRR)**，只反映已投入资金的真实表现，不被闲置现金稀释；买卖点标注成交净值、悬停可查看明细。
- **复盘分析**：区间收益、年化、最大回撤（含持续区间）、每笔交易盈亏、持有期浮动盈亏、相对基准超额。
- **多策略对比**：多套策略并排回测、指标与曲线对照。
- **VIX 恐慌指数模块**：内置 CBOE 真实 VIX 历史，滑动到任意日期显示当天真实值，辅助纳指基金择时。
- **VIX 自动交易策略**：设定日期范围与「VIX ＞X 买入 / ＜Y 卖出」阈值，在区间内自动生成买卖交易并回测（支持突破触发/持续区间触发）。

## 🚀 使用

1. 克隆或下载本仓库。
2. 直接双击 `index.html`（或用任意浏览器打开）即可，无需构建、无需服务器。

## 📊 数据来源

- **基金净值**：东方财富公开接口 `pingzhongdata`（`Data_netWorthTrend`），真实完整历史单位净值。
- **VIX 指数**：CBOE 官方 `VIX_History.csv`。
- 可通过 `build/gen_sample.py` 重新抓取并再生成 `js/sample-data.js`。
